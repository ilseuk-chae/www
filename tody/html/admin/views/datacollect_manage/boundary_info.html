    1. SHP 로드: 다운로드 shp 파일을 QGIS로 open 합니다.
    2. 인코딩: 레이어 마우스 오른쪽클릭 속성 → 인코딩(EUC-KR) → 확인(재실행해서 적용여부 확인)
    3. 재투영: 공간처리 툴박스(레이어 재투영:Reproject layer) 실행 → 입력레이어 선택,대상좌표계를 EPSG:4326 - WGS 84 선택→ 실행
    4. 도형 수정: 공간처리 툴박스(도형 수정:Fix geometries )→ 입력레이어(재투영된 산출물) 실행
    5. 단순화: 공간처리 툴박스(단순화:Simplify) 실행 → 방법(Douglas-Peucker),허용오차(emd:0.0001) → 실행
    6. 필드 재작성 : 공간처리 툴박스(필드 재작성:Refactor Fields ) 실행→입력레이어(수정한 도형) 이름(EMD_NM→EMD_KOR_NM)
    7. 레이어 분할 : 공간처리 툴박스(백터 레이어 분할:Split vector layer ) 실행→입력레이어(재작성 산출물), 유일ID필드(EMD_CD), 필드접두어추가 체크,산출물 파일 형(geojson), 출력폴더지정 → 실행

    1. 1~4 동일
    2. 필드계 산기: 공간처리 툴박스(필드 계산기) 실행→입력레이어(도형 수정) ,필드이름(SIG_CD), 필드길이(5),출력 필드 타입(Text (문자열)), 표현식(substr(to_string("EMD_CD"), 1, 5)) → 실행
    3. 융해 : 공간처리 툴박스(융해:Dissolve) 실행→ 입력레이어(산출물),융해필드(Dissolve field: SIG_CD) → 실행
    4. 단순화:공간처리 툴박스(단순화:Simplify) 실행 → 입력레이어(융해된 산출물) 방법(Douglas-Peucker),허용오차(emd:0.0001, 나머지는0.0003)) → 실행
    5. 도형 수정: 공간처리 툴박스(도형 수정:Fix geometries )→ 입력레이어(단순화 산출물) 실행
    6. 필드 재작성 : 공간처리 툴박스(필드 재작성:Refactor Fields ) 실행→입력레이어(수정한 도형) 이름(EMD_NM→SIG_KOR_NM, EMD_CD→SIG_ENG_NM, COL_ADM_SE, SGG_OID 삭제)

    8. 산출된 geojson 파일을 열어SIG_KOR_NM,SIG_ENG_NM 내용을 수정

            @echo off
            setlocal enabledelayedexpansion
            
            for %%f in (*.geojson) do (
                call :process "%%f"
            )
            goto :eof
            
            :process
            set "file=%~1"
            set "base=%~n1"
            
            set "code=!base:EMD_CD_=!"
            set "newcode=!code!00"
            
            echo Processing !file! → EMD_CD_!newcode!.geojson
            
            ogr2ogr -f GeoJSON "EMD_CD_!newcode!.geojson" "!file!" ^
            -dialect SQLite ^
            -sql "SELECT EMD_CD || '00' AS BJCD, EMD_KOR_NM AS NAME, geometry FROM \"!base!\""
            
            goto :eof
    6. convert.bat 실행
    7. EMD_CD_3pro.geojson 과 변환된 개별 EMD_CD_41591101.geojson QGIS에 로드
    8. 공간처리 툴박스(백터 레이어 병합:Merge vector layers ) 실행→입력레이어 선택, 병합 산출물 파일 저장 파일 형(geojson), 출력폴더, 이름지정

          @echo off
          setlocal enabledelayedexpansion

          for %%f in (SIG_CD_*.geojson) do (
              echo %%f | findstr /r "00000\.geojson" >nul
              if errorlevel 1 (
                  call :process "%%f"
              )
          )
          goto :eof

          :process
          set "file=%~1"
          set "base=%~n1"

          set "code=!base:SIG_CD_=!"
          set "newcode=!code!00000"

          set "outfile=SIG_CD_!newcode!.geojson"

          echo Processing !file! → !outfile!

          if exist "!outfile!" del "!outfile!"

          ogr2ogr -f GeoJSON "!outfile!" "!file!" ^
          -dialect SQLite ^
          -sql "SELECT SIG_CD || '00000' AS BJCD, SIG_KOR_NM AS NAME, geometry FROM \"!base!\""
