  • ☞ 주소 : https://www.vworld.kr/dtmk/dtmk_ntads_s002.do?svcCde=NA&dsId=23   연속지적도형 정보 바로가기
  • ☞ 시도별 다운로드 (시.도 전체, 전체구분(전체데이터), 전체(SHP) 선택) 가능
  • ☞ 좌표계 정보 포함
  • ☞ SHP 형식을 CSV로 변환 필요

<실행 순서>

  1. SHP 파일 로딩 — GIS 소프트웨어로 다운로드한 SHP 파일의 레이어를 불러옵니다.
  2. CSV 변환 — 불러온 레이어를 CSV 형식으로 변환합니다.
  3. DB 적재 — CSV 파일을 DB 테이블에 적재합니다.

<방법>

  1. QGIS로 다운받은 SHP 파일을 엽니다.
  2. 메뉴 공간처리 → 툴박스 → "레이어 재투영" 실행
    입력 레이어를 선택하고, 대상 좌표계를 EPSG:4326 - WGS 84로 선택 후 실행합니다.
  3. 레이어 "재투영된 산출물"에서 마우스 우클릭
    → 내보내기 → 피처를 다른 이름으로 저장 선택 후 아래 옵션 적용:
    • 포맷 : 쉼표로 구분된 값 [CSV]
    • 좌표계 (CRS) : EPSG:4326 - WGS 84
    • 인코딩 : UTF-8
    • 레이어 옵션 — GEOMETRY : AS_WKT, STRING_QUOTING : ALWAYS
    • 파일 이름 입력 후 확인 실행
  4. 저장된 CSV 파일을 AL_D002_tmp 테이블에 저장합니다.
    ※ 완료 후 AL_D002_tmp 의 WKT 값이 (127.xxx, 38.xxx) 형식인지 확인 
    -- 이전 테이블 삭제
        TRUNCATE TABLE AL_D002_tmp;
        
        -- CSV 파일 적재
        LOAD DATA LOCAL INFILE 'E:/source/Tody/AL_D002/AL_D002_41_20260308.csv'  -- 영문 경로로 변경
        IGNORE
        INTO TABLE AL_D002_tmp
        FIELDS TERMINATED BY ','
        OPTIONALLY ENCLOSED BY '"'
        LINES TERMINATED BY '\r\n'
        IGNORE 1 ROWS;
        
        -- 데이터 개수 확인
        SELECT COUNT(*) FROM AL_D002_tmp;
  5. AL_D002_tmp 을 AL_D002_Temp2 컬럼에 맞게 복사(저장된 CSV 파일이 여러개일 경우 반복수행) 
    
        INSERT INTO AL_D002_Temp2 (
            WKT,      -- GEOMETRY 타입 (내부적으로 이진 저장)
            A0, 
            pnu_cd,   -- tmp의 A1
            bjd_cd,   -- tmp의 A2
            A3,       -- 주소
            bunji,    -- tmp의 A4
            A5,       -- 지목/상태
            A6,       -- 날짜
            A7,       -- 지역코드
            layer, 
            path, 
            A10
        )
        SELECT 
            ST_GeomFromText(WKT, 4326), -- 텍스트를 이진 공간 데이터로 변환
            A0, 
            A1, 
            A2, 
            A3, 
            A4, 
            A5, 
            A6, 
            A7,
            NULL,     -- layer 초기값
            NULL,     -- path 초기값
            NULL      -- A10 초기값
        FROM AL_D002_tmp
        WHERE WKT IS NOT NULL AND WKT LIKE 'MULTI%';

        SELECT COUNT(*) FROM AL_D002_Temp2;
  6. AL_D002_Temp2 을 AL_D002_xx 컬럼에 맞게 복사
    
        SET autocommit = 0;

        INSERT INTO AL_D002_xx
        SELECT * FROM AL_D002_Temp2;
        
        COMMIT;
